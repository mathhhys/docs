## VS Code Marketplace

To download Softcodes, you simply have to : 

1. Open VS Code
2. Click the Extensions icon in the Side Bar or press `Ctrl+Shift+X` (Windows/Linux) or `Cmd+Shift+X` (macOS)
3. Search for "Softcodes"
4. Select "Softcodes" by Softcodes and click Install
5. Restart the extension if needed

After the complete installation, you will find the Softcodes icon in the sidebar of your editor

## Troubleshooting

### Extension Not Visible

- Restart your VS Code editor
- Verify Softcodes is well installed and enabled in Extensions
- Try disabling and re-enabling the extension in Extensions directly
- Check Output panel for errors ( View -\> Output, select "Softcodes")

### Installation Problems

- Ensure stable internet connection
- Verify VS Code version 1.84.0 or later
